HTML Terms

Hyper-text Markup Language

HTML element

An HTML element is a type of HTML document component.

HTML tag

HTML tags are used to hold the HTML element.

angle bracket

Commonly used to enclose a code of some type.

HTML attribute

HTML attributes are special words used inside the opening tag to control the element's behaviour.

HTML comment

HTML comments are visible to anyone that views the page source code, but are not rendered when the HTML document is rendered by a browser.

HTML object element

HTML object element represents an external resource

HTML special character

Since <, >, and & have special meanings in HTML, you can't use them to represent themselves in an HTML document. When you do want to use them literally (as in HTML source examples), you need to mark them up with special codes.
